# Option-Pricer

A tool for pricing financial options using various mathematical models.

## Features

- Price European and/or American options
- Supports Black-Scholes model
- Calculates option Greeks (Delta, Gamma, Theta, Vega, Rho)

## Installation

```bash
git clone https://github.com/Ra4ster/Option-Pricer.git
cd Option-Pricer
```
